How to start a project: 
create web server (only have to do once):
npm install -g http-server
sudo = run as administrator

everytime we run the project:
- Open a terminal 
- Navigate to project in terminal:
cd D:/Programming/......

-then run it with 
http-server -c-1

-point your browser at http://localhost:8080/


while working on a project:

Open a terminal, navigate to your project 

do this if you added new files:
git add . 

everytime you want to make changes to github:
git commit -am'commit messages goes here' 
git push
